# Introduction

**AGNX Drawer** is an AI-powered diagram creation tool that combines the power of Large Language Models with professional diagramming capabilities.

## What is AGNX Drawer?

AGNX Drawer allows you to create, modify, and enhance diagrams through natural language commands. Simply describe what you want, and the AI will generate the diagram for you.

## Key Features

- **Dual Canvas Engine**: Choose between Draw.io (professional/precise) and Excalidraw (hand-drawn/sketch) styles
- **Natural Language Interface**: Create diagrams by describing them in plain English
- **Multi-Provider Support**: Works with 10+ AI providers including OpenAI, Anthropic, Google, DeepSeek
- **Cloud Architecture Diagrams**: Specialized support for AWS, GCP, Azure architecture diagrams
- **DSL Support**: PlantUML for Draw.io, Mermaid for Excalidraw
- **MCP Integration**: Use with Claude Desktop, Cursor, VS Code via Model Context Protocol
