# 致谢

## 原始项目

AGNX Drawer 是基于 **[@DayuanJiang](https://github.com/DayuanJiang)** 创建的 [next-ai-draw-io](https://github.com/DayuanJiang/next-ai-draw-io) 进行二次开发的。

我们向原作者致以最深切的感谢：

- 创造了 AI 驱动图表创建的创新理念
- 构建了坚实、架构良好的基础
- 以 Apache 2.0 许可证开源项目
- 激励了开发者社区

**原始仓库**: [https://github.com/DayuanJiang/next-ai-draw-io](https://github.com/DayuanJiang/next-ai-draw-io)

## 赞助商

特别感谢[字节跳动豆包](https://console.volcengine.com/ark/)赞助演示站点的 API Token 使用。

## 开源库

本项目使用了以下优秀的开源软件：

- [Next.js](https://nextjs.org/) - React 框架
- [Vercel AI SDK](https://sdk.vercel.ai/) - AI 集成
- [react-drawio](https://github.com/marcveens/react-drawio) - Draw.io React 封装
- [Excalidraw](https://excalidraw.com/) - 手绘风格图表库
- [shadcn/ui](https://ui.shadcn.com/) - UI 组件
- [Tailwind CSS](https://tailwindcss.com/) - 样式框架
