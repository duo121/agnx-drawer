# 简介

**AGNX Drawer** 是一款 AI 驱动的图表创建工具，将大语言模型的强大能力与专业绘图功能相结合。

## 什么是 AGNX Drawer？

AGNX Drawer 让你通过自然语言命令创建、修改和增强图表。只需描述你想要的内容，AI 就会为你生成图表。

## 核心功能

- **双画板引擎**: 可选择 Draw.io（专业/精确风格）或 Excalidraw（手绘/草图风格）
- **自然语言界面**: 用简单的语言描述即可创建图表
- **多提供商支持**: 支持 OpenAI、Anthropic、Google、DeepSeek 等 10+ AI 提供商
- **云架构图**: 专门支持 AWS、GCP、Azure 架构图
- **DSL 支持**: Draw.io 支持 PlantUML，Excalidraw 支持 Mermaid
- **MCP 集成**: 通过模型上下文协议与 Claude Desktop、Cursor、VS Code 集成
