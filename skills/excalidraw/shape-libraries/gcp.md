# GCP Icons for Excalidraw

**Type:** Excalidraw library icons
**Category:** `gcp`

## Usage

```json
{"$icon": "gcp/compute_engine", "x": 100, "y": 100}
{"$icon": "gcp/cloud_storage", "x": 300, "y": 100}
{"$icon": "gcp/bigquery", "x": 500, "y": 100}
```
